RESPONSE:

Stripe is a company that provides a suite of tools for processing payments online. These tools include a payment gateway, a merchant account, and a set of APIs for developers to use in building their own applications. Stripe also offers a pre-built checkout page and a set of tools for managing subscriptions and invoices.
